# Placement Talk – MERN Stack Blogging Platform

A full-stack blogging platform built using the MERN (MongoDB, Express.js, React.js, Node.js) stack to facilitate structured placement-related discussions and content sharing.

## 🛠️ Tech Stack

- **Frontend:** React.js, React Router, Tailwind CSS
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **API Testing:** Postman
- **Version Control:** Git, GitHub
